# 🥜ィュ占い PWA (Progressive Web App)

iPhoneのホーム画面に追加できるWebアプリです!

## 📱 iPhoneへのインストール方法

### 1. Webサーバーにアップロード

すべてのファイルを同じフォルダにアップロード:
- `index.html` (メインファイル)
- `manifest.json` (アプリ設定)
- `sw.js` (Service Worker)
- `IMG_7110_2.JPG` (背景画像)
- `icon-192.png` (アプリアイコン 小)
- `icon-512.png` (アプリアイコン 大)

### 2. iPhoneでサイトにアクセス

Safari でアップロードしたURLを開く

### 3. ホーム画面に追加

1. 画面下部の共有ボタン(□↑)をタップ
2. 「ホーム画面に追加」を選択
3. アプリ名を確認して「追加」をタップ

### 完成! 🎉

ホーム画面に🥜のアイコンが追加され、アプリのように使えます!

## ✨ PWAの機能

- 📱 ホーム画面から起動

## 🌐 無料ホスティングサービス

以下のサービスなら無料で簡単にホスティングできます:

1. GitHubアカウント作成
2. リポジトリ作成

### Netlify
1. アカウント作成
2. フォルダをドラッグ&ドロップ

1. アカウント作成
2. プロジェクトをインポート
3. 自動デプロイ

## 🛠️ ローカルでテストする方法

### 方法1: Python（簡単!）
```bash
# ファイルがあるフォルダで実行
python3 -m http.server 8000

# ブラウザで開く
# http://localhost:8000
```

### 方法2: Node.js
```bash
# http-serverをインストール
npm install -g http-server

# 実行
http-server

# ブラウザで開く
# http://localhost:8080
```

### 方法3: VS Codeの拡張機能
- "Live Server" 拡張機能をインストール
- index.htmlを右クリック > "Open with Live Server"

## 🔧 トラブルシューティング

**Q: ホーム画面に追加できない**
- SafariでHTTPS接続が必要です
- GitHub Pages等の無料ホスティングはHTTPS対応済み

**Q: オフラインで動かない**
- 一度オンラインでアクセスしてキャッシュを作成する必要があります

**Q: アイコンが表示されない**
- ブラウザのキャッシュをクリアして再度追加してみてください
